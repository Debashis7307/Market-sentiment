Project: Trader Behavior vs Market Sentiment

Overview

This project analyzes how Bitcoin market sentiment (Fear/Greed Index) impacts trader behavior and profitability using Hyperliquid historical trading data.

Key Objectives
Understand relationship between sentiment and PnL
Identify behavioral biases in traders
Derive actionable trading strategies

Dataset
Bitcoin Fear & Greed Index
Hyperliquid Trader Data

Key Findings
Traders perform better during Fear phases
Overtrading and high leverage during Greed reduce profitability
Top traders follow contrarian strategies

Strategies Derived
Sentiment-aware trading system
Leverage risk control
Smart trader tracking
Contrarian trading model

Tech Stack
Python
Pandas
Matplotlib / Seaborn

How to Run
pip install pandas matplotlib seaborn
python analysis.py
